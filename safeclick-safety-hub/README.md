# SafeClick Safety Hub

A responsive React prototype featuring:

- Splash screen
- Three onboarding screens
- Safety Hub home screen
- Activity timeline
- Care Network
- Profile and interactive toggles
- Responsive phone frame on desktop
- Full-screen mobile layout
- Smooth transitions and animated protection ring

## Run locally

```bash
npm install
npm run dev
```

## Build for deployment

```bash
npm run build
```

## Upload to GitHub

1. Create a new GitHub repository.
2. Upload all files from this project.
3. Commit the files.
4. Deploy using Vercel, Netlify, or GitHub Pages.

For Vercel or Netlify, import the GitHub repository and use:

- Build command: `npm run build`
- Output folder: `dist`
